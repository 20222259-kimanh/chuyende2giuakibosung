# English Learning App

A premium Flutter application for learning English online.

## Features
- **Modern Dashboard**: Welcome screen with a "Lesson of the day" feature.
- **Lesson List**: High-quality cards showing lesson previews, duration, and ratings.
- **Detailed Lesson View**: Immersive header, lesson description, and structured learning points.
- **Modern Aesthetics**: Uses Google Fonts (Outfit), gradients, and Material 3 design.

## How to Run
1. Ensure you have [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
2. Navigate to the project directory:
   ```bash
   cd english_learning_app
   ```
3. Get dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## Project Structure
- `lib/main.dart`: App entry point.
- `lib/models/lesson.dart`: Data models and dummy content.
- `lib/screens/lesson_list_screen.dart`: List of all available lessons.
- `lib/screens/lesson_detail_screen.dart`: Individual lesson details.
